# WdkMvpPlugin
```
使用方式：
直接把jar包导入到Android studio插件设置中，重新启动Android studio
在创建文件的时候new右菜单底部会出现WdkMvpPlugin,点击WdkMvpPlugin出现弹框类似创建文件的弹框
注意输入框有定义规则
data层：xxxdatamvp         例如：Logindatamvp
viewmodel层：xxxviewmvp    例如：Loginviewmvp
Activity创建：xxxActivity  例如：LoginActivity
Fragment创建：xxxFragment  例如：LoginFragment
```
